# 🩺 Medical Chatbot (Groq API)

An **AI-powered Medical Chatbot** built using **HTML, CSS, JavaScript, and the Groq API**.  
This chatbot allows users to ask health-related questions and receive AI-generated responses about symptoms, diseases, and treatments in a simple chat interface.

⚠️ **Note:** This chatbot provides general medical information only and should not replace professional medical advice.

---

# 🚀 Features

- 💬 Interactive chat interface
- ⚡ Fast responses using Groq API
- 🧠 Powered by LLaMA 3.1 AI model
- 🎨 Clean dark UI design
- 📱 Simple and responsive layout
- 🩺 Answers basic medical questions
- 🔒 Encourages consulting healthcare professionals

---

# 🛠 Technologies Used

- HTML
- CSS
- JavaScript
- Groq API
- LLaMA 3.1 AI Model

# ⚙️ Setup Instructions

## 1️⃣ Clone the Repository

```bash
git clone https://github.com/rsamwilson2323-cloud/medical-chatbot.git
```

---

## 2️⃣ Open the Project

Open the folder and run:

```
index.html
```

in your browser.

---

## 3️⃣ Add Your Groq API Key

Find this line in the code:

```javascript
const GROQ_API_KEY="PASTE_YOUR_GROQ_KEY_HERE";
```

Replace it with your Groq API key.

Example:

```javascript
const GROQ_API_KEY="gsk_your_api_key_here";
```

---

# 💡 How It Works

1. User enters a medical question
2. The message is sent to the Groq API
3. The LLaMA AI model processes the question
4. The AI generates a response
5. The response appears in the chatbot interface
